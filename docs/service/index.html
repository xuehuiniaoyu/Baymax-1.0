Android Baymax框架 - Service

1.原生的Service构建方式

和Activity一样，Service要想使用必须在AndroidManifest.xml中配置。

代码：
public class TestService extends Service {

    ...

}

需要在AndroidManifest.xml中配置

<service android:name=".TestService" />

如果我们不在AndroidManifest.xml中进行配置缺直接通过startService(new Intent(context, TestService.class)); 系统可能不会抛出异常，但是不能够使用。

2.Baymax框架中Service的构建方式

@Tag_Service(name = "test-service")
public class TestService extends Service {

     ...

}

注意：不需要在AndroidManifest.xml中配置。

启动TestService有两种方式：

1. startService(new Intent(context, TestService.class));

2. Baymax.single().service("test-service").start();
